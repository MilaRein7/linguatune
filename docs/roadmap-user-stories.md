# Дорожная карта разработки документации проекта LinguaTune

В данном документе приведена дорожная карта разработки документации для проекта **LinguaTune**.  
Структура: **Эпики → Пользовательские истории → Задачи по документации**.

---

## Эпик 1. Установка и запуск LinguaTune

### User Story 1.1 — Установка и запуск через консоль

> Как новый пользователь,  
> я хочу видеть в документации требования и пошаговые инструкции по установке и запуску через командную строку,  
> чтобы быстро и без ошибок запустить приложение на своём компьютере.

**Задачи по документации:**

1. В `README.md` описать:
    - версии JDK и Maven;
    - поддерживаемые операционные системы;
    - команды:
        - `git clone https://github.com/MilaRein7/linguatune.git`;
        - `cd linguatune`;
        - `mvn clean javafx:run`.
2. Добавить пояснение, что Maven автоматически скачает зависимости.
3. В `docs/quick-start.md` сделать короткую инструкцию «Установка и запуск в 3 шага».
