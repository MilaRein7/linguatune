# LinguaTune — учим английский по песням

Учебный JavaFX-проект для практики английского языка с помощью песен.  
Приложение позволяет выбирать песню из каталога, кликать по словам в тексте, смотреть перевод и добавлять новые слова в личный словарь.

---

## 1. Описание проекта

**LinguaTune** — это настольное приложение для изучающих английский язык (уровни A1–C1), которое помогает пополнять словарный запас на основе любимых песен.

Основные возможности:

- Каталог песен с указанием исполнителя, жанра и уровня сложности.
- Фильтрация песен по уровню (A1–C1) и поиску по названию / исполнителю.
- Просмотр текста песни в отдельном окне.
- Клик по слову в тексте → показ перевода (из базового словаря) и предложение добавить его в личный словарь.
- Вкладка **«Словарь»** — список всех сохранённых слов и их переводов.
- Экспорт словаря в CSV-файл.
- Вкладка **«Профиль»** — имя пользователя и мини-статистика (количество песен и слов в словаре).

Проект создан как учебный в рамках практики по Java / JavaFX и оформлению технической документации.

## 2. Установка и запуск

### 2.1. Требования

Перед запуском убедитесь, что установлено:

- **Java JDK**: 17+ (проект тестировался на JDK 25)
- **Maven**: 3.8+
- **Git** (если клонируете репозиторий по HTTPS/SSH)

ОС: Windows / macOS / Linux (JavaFX кроссплатформенный).

---

### 2.2. Клонирование репозитория

```bash
git clone https://github.com/MilaRein7/linguatune.git
cd linguatune
```

---

### 2.3. Запуск через Maven (консоль)

Из корня проекта выполните:

```bash
mvn clean javafx:run
```

Maven скачает зависимости (JavaFX и др.) и запустит приложение.

---

### 2.4. Запуск в IntelliJ IDEA

1. Открыть **IntelliJ IDEA** → `File` → `Open…`.
2. Выбрать папку проекта `linguatune` (или `pom.xml`).
3. Дождаться импорта Maven-проекта.
4. Найти класс `com.linguatune.App`.
5. Запустить `App.main()` (зелёная кнопка **Run**).

---

## 3. Примеры использования

### 3.1. Работа с профилем

1. Перейдите на вкладку **«Профиль»**.
2. Введите ваше имя и нажмите **«Сохранить профиль»**.
3. Имя будет отображаться в профиле, а статистика покажет:
   - количество песен в каталоге;
   - количество слов в словаре.

Данные профиля сохраняются в локальный файл `profile.txt` (не попадает в репозиторий).

---

### 3.2. Каталог песен

1. Откройте вкладку **«Каталог»**.
2. Используйте поле **«Поиск»** для фильтрации по названию или исполнителю.
3. Выберите уровень в выпадающем списке (A1–C1 или «Все уровни»).
4. Дважды кликните по песне в таблице — откроется окно с текстом.

---

### 3.3. Добавление слов в словарь

1. В открытом окне с текстом песни кликните по любому английскому слову.
2. Если слово найдено в базовом словаре:
   - будет показан перевод;
   - появится вопрос, добавить ли слово в личный словарь.
3. При подтверждении слово добавится во вкладку **«Словарь»**.

Личный словарь хранится в локальном файле `user-dictionary.txt`.

---

### 3.4. Работа со словарём

Во вкладке **«Словарь»**:

- отображаются все добавленные слова и их перевод;
- можно удалить выбранное слово;
- можно экспортировать словарь в CSV-файл:
  - нажмите **«Скачать словарь (CSV)»**;
  - выберите путь и имя файла (по умолчанию `linguatune-dictionary.csv`);
  - файл сохраняется в кодировке `Windows-1251`, разделитель — `;`.

## 4. Структура репозитория

Краткое описание основных директорий и файлов:

```text
linguatune/
├── pom.xml                       # Maven-конфигурация проекта
├── .gitignore                    # Настройки игнорирования файлов Git
└── src/
    └── main/
        ├── java/
        │   └── com/
        │       └── linguatune/
        │           ├── App.java              # Точка входа JavaFX-приложения
        │           ├── controller/
        │           │   └── MainController.java  # Главный контроллер UI
        │           ├── model/
        │           │   ├── Song.java         # Модель песни
        │           │   ├── Playlist.java     # Модель плейлиста
        │           │   └── Word.java         # Модель слова словаря
        │           └── service/
        │               └── SongRepository.java  # Загрузка тестовых песен
        └── resources/
            └── com/
                └── linguatune/
                    ├── view/
                    │   ├── main-view.fxml    # Разметка интерфейса (FXML)
                    │   └── style.css         # Стили оформления
                    └── data/
                        └── base-dictionary.txt  # Базовый словарь (англ–рус)
```

Локальные файлы `profile.txt` и `user-dictionary.txt` создаются приложением при запуске и не входят в репозиторий.

## 5. Технические требования

- **Язык разработки**: Java  
- **GUI**: JavaFX  
- **Сборка**: Maven (`javafx-maven-plugin`)  
- **Версия JDK**:  
  - минимально рекомендуется **Java 17+**

- **Операционные системы**:
  - Windows 10/11  
  - macOS  
  - Linux (при наличии соответствующей сборки JavaFX)

Для разработки удобно использовать **IntelliJ IDEA Community Edition** (или любую IDE с поддержкой Maven/JavaFX).
