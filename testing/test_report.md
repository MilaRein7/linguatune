## 3. Результаты пользовательского тестирования

Проведено короткое тестирование с 3 одногруппниками:

- Участник 1: новичок в Java/IDE, но уверенный пользователь ПК.
- Участник 2: часто пользуется скриптовыми языками, редко — настольными приложениями.
- Участник 3: обычный пользователь без технического бэкграунда.

Каждому был предложен сценарий U1 (новый пользователь).  
Обратная связь собиралась через Google Forms (см. `feedback_summary.md`).

**Наблюдения:**

- Все участники смогли:
    - задать имя в профиле;
    - открыть песню из каталога;
    - добавить хотя бы одно слово в словарь;
    - найти список слов во вкладке «Словарь».
