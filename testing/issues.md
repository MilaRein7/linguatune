## 1. Ошибки / ограничения

### Issue 1 — Пояснение по уровню песен (A1–C1)

- **Тип:** Улучшение UX / содержимое
- **Источник:** Опрос пользователей
- **Описание:** Некоторым пользователям не вполне понятно,
  чем отличаются уровни A1–C1.
- **Предложение:**
    - добавить короткую подсказку рядом с фильтром уровня;
    - либо отдельный раздел в User Guide с кратким описанием уровней CEFR.
- **Приоритет:** Низкий
